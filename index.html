Details:

Big Data provides the flexibility to store a large dataset without any constraints to the type of the data. Our proposed Health Recommendation System(HRS) is being implemented using Big Data features.
Yelp dataset is selected for the implementation of this recommendation system. Yelp dataset contains all the information pertaining to hospitals, doctors, ratings and feedback of the patients (reviews) based on geographical location. The main purpose is practical implementation of Big Data concepts that are being taught in the class.

Dataset used :
Dataset source: Yelp Academic Dataset
Data Subset used: (1) business.json (109MB) (2) review.json (3.22GB)
Methodology Followed:
Data Cleaning and Analysis:

  • Initially downloaded the yelp dataset.

  • Extracted downloaded dataset to obtain JSON files.

  • Converted JSON files into more readable format like csv/tsv using python script and pig script respectively

  • According to the problem statement, confined our analysis to review and business datasets.

  • Using Python script, cleaned and analyzed the datasets.

Data Cleaning and Analysis:

  • Created hive tables with the relevant fields for each dataset (business and review) with respect to the problem statement.

  • Performed NLTK processing on review text field and added new column to review.csv file to distinguish between positive, negative and neutral reviews.

  • Creating new tables by integrating required fields from review and business tables using hive query.

  • New tables where created according to positive reviews, negative reviews, specialization of doctors based on city/state.

  • Above created tables were converted into csv format and was visualized in tableau..
